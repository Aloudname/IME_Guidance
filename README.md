## 引言

> "生物医学工程究竟是个什么样子，高教界搞了很多年，也并没有完全搞清楚。可能一些高校的思路比较好，搞了个智能医学工程，可惜后来也僵化了。"

这个仓库面向以 **智能医学工程专业生** 身份踏入大学校园的学弟学妹们，为羽翼还未丰满、对未来还不甚了解的你们，提供求学、工作和生活上的微薄参考。

作为 **"新兴专业"** 的前几批学生，我们成长求学的道路上少不了碰壁、挫折和困惑。如今大学四年生活已近尾声，各奔前程之时，我们对曾经的一些问题有了自己的答案，也逐渐发现了目前生涯规划中的不足。于是，我们想尽可能多地将自己来时路上的挫折和感悟记录下来，给你们讲述自己的 **"失败学"**。

我们感谢四年时光里给予我们无限支持和关怀的老师和辅导员，感谢培育我们的山东大学控制科学与工程学院。

## 这个 repo 有什么？

我们提供一切有关求学、工作、生活上的微薄参考，分成三个部分：

<div align="center">

| | 模块 | 内容 |
|:-:|:--|:--|
| 1 | **基础技能** | 信息检索、GitHub 使用、AI 工具等现代工程师必备技能 |
| 2 | **学业 & 工作** | AI 全栈职业规划、培养方案解读、市场需求分析 |
| 3 | **生活** | 校园生活经验、心态调整、避坑指南 |

</div>

重点分享的内容是 **学业** 和 **工作**。作为**电子电气大类**的学生，我们需要具备相当完备的计算机知识和素养——这些都是目前培养方案欠缺的。**一定不要完全依赖培养方案，一定要有自己的打算。**

## 贡献指南

欢迎通过以下方式让这个仓库变得更好：

- **反馈问题** → 在 [issues](../../issues) 提交 bug 报告或功能建议
- **贡献内容** → fork 本仓库，做出修改后提交 [pull request](../../pulls)

新内容请使用 Markdown 格式，图片资源放在 `src/` 对应目录下，保持本仓库的风格和分册结构。
